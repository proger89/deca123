# Веб-демо SafeSort

Статическая русскоязычная версия для жюри. STL разбирается локально в браузере и не отправляется на сервер.

```powershell
python -m http.server 8080 --directory web
```

Откройте `http://localhost:8080`, загрузите ASCII или binary STL и нажмите «Проверить модель».

Проверка парсера, строгих границ и устойчивости к повороту синтетических STL:

```powershell
node --test tests/web/app_geometry.test.cjs
```

Демонстратор уточняет ориентированный ограничивающий параллелепипед поиском меньшего объёма, строит адаптивные сечения по его осям и доминирующим нормалям поверхности и оценивает `K = r/R` по контуру сечения. Затем применяется официальный порядок правил: сначала строгие габариты, затем `K`. Это объяснимый браузерный предпросмотр; итоговое доказательство многовидового восприятия, механики заслонок и подтверждения выхода выполняет Webots-сценарий из корня проекта.

Состояния для проверки интерфейса:

- `?state=loading`
- `?state=partial`
- `?state=fault`
- `?state=tampered`

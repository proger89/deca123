# Веб-демо SafeSort

Статическая русскоязычная версия для жюри. STL разбирается локально в браузере и не отправляется на сервер.

```powershell
python -m http.server 8080 --directory web
```

Откройте `http://localhost:8080`, загрузите ASCII или binary STL и нажмите «Проверить модель».

Демонстратор выполняет PCA-оценку ориентированного ограничивающего параллелепипеда, оценивает `K = r/R` на проекциях и применяет официальный порядок правил: сначала строгие габариты, затем `K`. Это объяснимый браузерный предпросмотр. Финальное доказательство многовидового восприятия, механики заслонок и подтверждения выхода выполняет Webots-сценарий из корня проекта.

Состояния для проверки интерфейса:

- `?state=loading`
- `?state=partial`
- `?state=fault`
- `?state=tampered`
